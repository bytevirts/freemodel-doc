title: "Billing"
description: "Manage prepaid balance and usage."
---

FreeModel uses prepaid USD balance. There are no subscriptions.

## Add balance

In the [FreeModel Console](https://freemodel.app/settings), open **Billing** and choose a top-up pack: `$5`, `$20`, `$50`, or `$100`.

## How usage is charged

Each request is metered against your balance. Costs depend on the selected model and, where applicable, the input and output tokens or generated media.

View the [models page](https://freemodel.app/models) for current per-model prices. View request logs and usage in the Console to understand recent consumption.

If your balance cannot cover a request, the API returns `insufficient_user_quota`. Add funds before retrying.

For refund terms, read the [FreeModel refund policy](https://freemodel.app/refund-policy).
