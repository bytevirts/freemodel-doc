Seedance 2 is available through the OpenAI-compatible Videos API. Both routes support **5, 10, and 15-second** clips in landscape, portrait, and square formats.

| Model ID            | Duration         | Price         |
| ------------------- | ---------------- | ------------- |
| `video-ds-2.0`      | 5, 10, or 15 sec | $1, $2, or $3 |
| `video-ds-2.0-fast` | 5, 10, or 15 sec | $1, $2, or $3 |

Prices correspond to 100, 200, and 300 credits. Seedance accepts `aspect_ratio`, not a raw pixel `size`. Output is 720p; a completed 16:9 result was verified at **1280×720**.

## Submit a Fast request

```bash
curl https://api.freemodel.app/v1/videos \
  -H "Authorization: Bearer $FREEMODEL_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "video-ds-2.0-fast",
    "prompt": "A paper sculpture unfolding into a city skyline, locked camera, clean studio light",
    "seconds": "5",
    "aspect_ratio": "16:9"
  }'
```

Save the returned `id`, then include the same model while polling and downloading:

```bash
curl "https://api.freemodel.app/v1/videos/$VIDEO_ID?model=video-ds-2.0-fast" \
  -H "Authorization: Bearer $FREEMODEL_API_KEY"

curl -L "https://api.freemodel.app/v1/videos/$VIDEO_ID/content?model=video-ds-2.0-fast" \
  -H "Authorization: Bearer $FREEMODEL_API_KEY" \
  --output video.mp4
```

See [Video generation](/api-reference/video-generation) for the full asynchronous workflow.
