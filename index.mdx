FreeModel is a unified AI API gateway. Use one API key and a familiar API shape to build with the models available to your account.

The primary API is OpenAI-compatible at `https://api.freemodel.app/v1`. Compatibility endpoints for Claude and Gemini are also available when your integration needs them.

## Start in three steps

<Steps>
  <Step title="Create your account">
    [Sign up for FreeModel](https://freemodel.app/sign-up) and open the [Console](https://freemodel.app/settings).
  </Step>
  <Step title="Add balance">
    In the Console, open **Billing** and add prepaid USD balance. FreeModel does not use subscriptions.
  </Step>
  <Step title="Create an API key">
    In the Console, open **API Keys** and create a key. Send it as `Authorization: Bearer sk-...` with every API request.
  </Step>
</Steps>

## Choose an API surface

Use the OpenAI-compatible API for new integrations. It supports chat completions, embeddings, images, model discovery, and server-sent event streaming. Use the Claude and Gemini compatibility endpoints when you are migrating an existing integration that expects those request formats.

<Note>
  Models and availability can change. Call [`GET /v1/models`](/api-reference/list-models) with your API key to discover models available to you. The [models page](https://freemodel.app/models) is the source of truth for current pricing.
</Note>
