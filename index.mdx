FreeModel is a unified AI API gateway. Use one API key and OpenAI-compatible endpoints to build with the models available to your account.

The API base URL is `https://api.freemodel.app/v1`. The live catalog currently includes GPT-5.4+ text models, GPT Image 2 image models, Gemini Omni video, and Seedance 2 video.

## Start in three steps

<Steps>
  <Step title="Create your account">
    [Sign up for FreeModel](https://freemodel.app/sign-up) and open the
    [Console](https://freemodel.app/settings).
  </Step>
  <Step title="Add balance">
    In the Console, open **Billing** and add prepaid USD balance. FreeModel does
    not use subscriptions.
  </Step>
  <Step title="Create an API key">
    In the Console, open **API Keys** and create a key. Send it as
    `Authorization: Bearer sk-...` with every API request.
  </Step>
</Steps>

## Choose an API surface

Use the OpenAI-compatible API for new integrations. The current catalog supports chat completions, image generation, asynchronous video generation, model discovery, and server-sent event streaming for compatible text models.

<Note>
  Models and availability can change. Call [`GET
  /v1/models`](/api-reference/list-models) with your API key to discover models
  available to you. The [models page](https://freemodel.app/models) is the
  source of truth for current pricing.
</Note>
